Ans 1. Set is a collection of unique data. We use set when we need to ensure that an item only appear once and we cannot use mixed datatypes in a set.

Syntax for creating a set is-
var VariableName : Set = [value1,value2,value 3]
Ex-

var num : Set = [1,2,3]
for i in num{
    print(i)
}
Output- 1 2 3


Main functions of set are-
VariableName.insert(value) -                        Insert an element in a set                          
VariableName.remove(value)-                      Remove an Element from a Set
Sorted()-				   sorts set elements
forEach()-                                                    performs the specified actions on each element
contains()-				searches the specified element in a set
randomElement()-			returns a random element from the set
firstIndex()-				returns the index of the given element
removeFirst() -				 to remove the first element of a set
removeAll() - 				to remove all elements of a set






Ans2. A range is a series of values between two numeric intervals.
Ex- var numbers= 1…4
Here, … is a range operator, 1 is lower bound and 4 is upper bound

	Types of ranges
a) Closed Range- Includes all values in the interval from lower bound to the upper bound.
Ex- for numbers in 1...4 {
  print(numbers)
}
Output- 1 2 3 4

b) Half-Open range- Includes all the values from the lower bound to the upper bound excluding the upper bound 
Ex- for numbers in 1..<4 {
  print(numbers)
}
Output- 1 2 3

c) One-sided Range- A one-sided range contains elements up to infinite in one direction.
Ex- let range=2… , It contains all elements from 2 to infinite.
